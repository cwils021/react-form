# React-Form

## To Run

---

### With Yarn (or replace yarn with npm)

```{shell}
$ yarn install
$ yarn run
```

### With Docker

**Ensure you have a .env with PORT var set before running (see .example.env)**

```{shell}
$ docker compose up -d -V
```
